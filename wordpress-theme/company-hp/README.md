# 会社HP テーマ（WordPress）

Lightsail 上の WordPress 用テーマです。React 版と同じデザイン（ヘッダー・フッター・ヒーロー・セクション）を PHP で再現しています。

## 必要な環境

- WordPress 5.0 以上
- 親テーマ不要（スタンドアロン）

## インストール

1. この `company-hp` フォルダを WordPress の `wp-content/themes/` に配置
2. 管理画面 → 外観 → テーマ で「会社HP（屋号）」を有効化
3. 表示設定で「ホームページの表示」を「固定ページ」にし、トップ用の固定ページを指定（任意）
4. 次の固定ページを追加し、表のスラッグにすると専用テンプレートが使われます（本文が空でもデフォルト文が表示されます）。
   | ページ名 | スラッグ |
   |----------|----------|
   | プライバシーポリシー | `privacy-policy` |
   | 利用規約 | `terms` |
   | 事業者について | `about` |
   | お問い合わせ | `contact` |
5. 外観 → メニュー で「メインメニュー」にページを追加

## 法令・慣例に沿った表示

フッターには「プライバシーポリシー」「利用規約」へのリンクが表示されます。次のスラッグで固定ページを作成するとリンクが有効になります。

- プライバシーポリシー: スラッグ `privacy-policy`
- 利用規約: スラッグ `terms`

「事業者について」のページには、屋号・事業者氏名・所在地・連絡先・個人事業主である旨を記載することを推奨します。詳細はリポジトリの `docs/LEGAL_DISPLAY.md` を参照してください。

## ファビコン・ホーム画面アイコン

テーマに **Axiola** をイメージしたアイコンが含まれています。

- **favicon.svg** … ブラウザタブ用（SVG）
- **apple-touch-icon.png** … スマホで「ホーム画面に追加」したときに使われるアイコン（180x180 / 192x192 として参照）

上記を `link` タグで出力しているため、スマホのホーム画面に追加した場合もアイコンが問題なく表示されます。管理画面で**外観 → カスタマイズ → サイト基本情報 → サイトアイコン**を設定すると、そちらが優先されます。

## トップ画像

テーマ内の **`assets/images/`** にトップ用画像を置きます。

- ファイル名: **hero.webp**（推奨）または **hero.jpg**
- 推奨サイズ: 1920 × 600 ～ 800px、200 ～ 500KB 程度

画像を置くとトップページのヒーローに表示されます。未配置の場合はテキストのみ表示されます。詳細は `assets/images/README.md` を参照してください。

## デプロイ（Lightsail）

リポジトリの `.github/workflows/deploy.yml` を WordPress テーマ用に設定し、`wordpress-theme/company-hp/` の中身を Lightsail の次のパスに rsync します。

- Bitnami: `/opt/bitnami/wordpress/wp-content/themes/company-hp/`
- 通常の WordPress: `~/wordpress/wp-content/themes/company-hp/`

詳細はリポジトリ直下の `docs/LIGHTSAIL_DEPLOY.md` を参照してください。
